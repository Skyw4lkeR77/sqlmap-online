Flags SQLMap

Berikut adalah beberapa flags dasar yang aman digunakan di panel ini untuk belajar SQL Injection:

Flag Penjelasan
--dbsMendaftar semua database yang tersedia
--tables -D [nama_db]Mendaftar tabel pada database tertentu
--columns -D [db] -T [tabel]Mendaftar kolom pada tabel tertentu
--dump -D [db] -T [tabel]Mendump isi tabel (data)
--dump-allMendump semua tabel di semua database (hati-hati, lama)
--risk=1 --level=1Tingkat pengujian dasar (default)
--risk=2 --level=2Pengujian lebih agresif (masih aman di sini)
--technique=BHanya gunakan teknik Boolean-based blind
--technique=THanya gunakan teknik Time-based blind
--technique=EHanya gunakan teknik Error-based
--technique=UHanya gunakan teknik Union query
--batchMode non-interaktif (sudah otomatis digunakan)
--flush-sessionMembersihkan cache session sebelum scan
--threads=3Menggunakan 3 thread (maksimal aman di demo)

Catatan: Flags berbahaya seperti --os-shell, --sql-shell, --file-read, dll. telah diblokir untuk keamanan.
